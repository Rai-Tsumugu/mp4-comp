# MP4圧縮ツール (mp4-comp)

このツールは、指定されたMP4動画ファイルを特定のサイズ（デフォルト：200MB以下）に自動的に圧縮するためのPythonスクリプトです。

## 特徴
- **サイズ目標指定**: 動画を特定のファイルサイズ（例：Discordなどのアップロード制限）に収まるようにビットレートを自動計算します。
- **2パスエンコード**: 高品質を維持しながら、正確なファイルサイズに近づけるために2パスエンコード方式を採用しています。
- **簡単操作**: スクリプトを実行してファイルをドラッグ＆ドロップするだけで圧縮が開始されます。

## 必要条件
このツールを実行するには、以下の環境が必要です。

1. **Python 3.x**
2. **FFmpeg**: システムのパス（PATH）に通っている必要があります。
   - `ffmpeg` および `ffprobe` コマンドがコマンドプロンプトやターミナルから実行できることを確認してください。

## 使い方

### 1. 手動実行（ドラッグ＆ドロップ）
1. `compress.py` を実行します。
   ```bash
   python compress.py
   ```
2. プロンプトが表示されたら、圧縮したいMP4ファイルをウィンドウにドラッグ＆ドロップするか、パスを直接入力してEnterキーを押します。
3. 続けてターゲットサイズ（MB）の入力を求められます。そのままEnterを押すとデフォルトの200MBが適用されます。

### 2. コマンドライン引数での実行
引数として「ファイルパス」と「ターゲットサイズ」を渡すことも可能です。
```bash
python compress.py "C:\path\to\your\video.mp4" 100
```
※第2引数を省略した場合は200MBが適用されます。

## 出力
- 圧縮されたファイルは、元のファイルと同じディレクトリに `_compressed` という接尾辞が付いた名前で保存されます。
- 例: `video.mp4` -> `video_compressed.mp4`

## 仕様の詳細
- **ターゲットサイズ**: デフォルトで200MBに設定されています（安全マージンとして実質195MB程度を目指します）。
- **音声**: 128kbps (AAC) 固定。
- **映像コーデック**: H.264 (libx264)。
- **計算式**: `(ターゲットサイズ - マージン) / 動画時間 = 合計ビットレート`

## 注意事項
- 非常に長い動画（数時間など）を200MBに圧縮しようとすると、ビットレートが極端に低くなり画質が著しく低下します。
- エンコード時間は動画の長さやPCのスペックに依存します。
- 実行中に一時的なログファイル（`ffmpeg2pass-0.log` など）が生成されますが、完了後に自動で削除されます。
